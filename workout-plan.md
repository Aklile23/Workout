# Weekly Workout Plan

A 4-day upper/lower split (Mon, Tue, Thu, Fri) focused on serious muscle growth (not maintenance). Saturday is an optional bonus day. Wednesday and Sunday are rest days.

---

## Monday — Upper A: Chest + Back Heavy

> This is not maintenance. This is a serious upper-body session.

| Exercise | Sets x Reps |
| --- | --- |
| Barbell bench press or machine chest press | 4 x 5–8 |
| Weighted pull-up / assisted pull-up / lat pulldown | 4 x 6–10 |
| Incline dumbbell press | 3 x 8–12 |
| Chest-supported row | 4 x 8–12 |
| Pec deck or cable fly | 3 x 12–15 |
| Lateral raise | 4 x 12–20 |
| Triceps pushdown | 3 x 10–15 |
| Dumbbell/cable curl | 3 x 10–15 |

---

## Tuesday — Lower A: Quad Focus

> This gives enough quad volume to grow.

| Exercise | Sets x Reps |
| --- | --- |
| Squat / hack squat / pendulum squat | 4 x 6–10 |
| Leg press | 3 x 10–15 |
| Romanian deadlift | 3 x 8–12 |
| Leg extension | 4 x 12–20 |
| Seated or lying leg curl | 3 x 10–15 |
| Standing calf raise | 4 x 8–15 |
| Cable crunch / hanging leg raise | 3 x 10–15 |

---

## Thursday — Upper B: Shoulders + Back + Arms

> This day fixes what many upper/lower programs undertrain: delts and arms.

| Exercise | Sets x Reps |
| --- | --- |
| Incline barbell/dumbbell press | 3 x 8–12 |
| Seated cable row | 4 x 8–12 |
| Lat pulldown, different grip from Monday | 3 x 10–12 |
| Machine shoulder press | 3 x 8–12 |
| Cable lateral raise | 4 x 12–20 |
| Rear delt fly / reverse pec deck | 4 x 12–20 |
| Overhead triceps extension | 3 x 10–15 |
| Preacher curl / incline curl | 3 x 10–15 |

---

## Friday — Lower B: Posterior Chain + Full Legs

> This is hard. You will not look like someone "maintaining" if you train this properly.

| Exercise | Sets x Reps |
| --- | --- |
| Romanian deadlift / deadlift variation | 4 x 6–10 |
| Hack squat / front squat / leg press | 3 x 8–12 |
| Hip thrust or glute bridge | 3 x 8–12 |
| Seated or lying leg curl | 4 x 10–15 |
| Bulgarian split squat / walking lunge | 3 x 8–12 each leg |
| Calf raise | 4 x 10–15 |
| Plank / cable crunch | 3 sets |

---

## Saturday — Optional Bodybuilding Day

This is where you make it more bodybuilding-focused without destroying your week.

**Pick ONE version.**

### Option A — Arms + Delts

| Exercise | Sets x Reps |
| --- | --- |
| Cable lateral raise | 4 x 15–25 |
| Rear delt fly | 3 x 15–25 |
| Triceps pushdown | 3 x 12–20 |
| Overhead triceps extension | 3 x 12–20 |
| Cable curl | 3 x 12–20 |
| Hammer curl | 3 x 12–20 |

### Option B — Chest + Arms

| Exercise | Sets x Reps |
| --- | --- |
| Incline machine press | 3 x 10–15 |
| Cable fly | 3 x 12–20 |
| Dips / assisted dips | 3 x 8–12 |
| Triceps pushdown | 3 x 12–20 |
| Preacher curl | 3 x 10–15 |
| Hammer curl | 3 x 12–15 |

### Option C — Weak Point Day

Use this if one body part is clearly lagging:

- Delts
- Arms
- Upper chest
- Calves
- Back width

> Keep Saturday to **45–70 minutes**. It is a pump / volume day, not another full war.
